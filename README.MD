# Microfilm
### Personal newspaper preservation
Archive news articles, save them as simple HTML files with detailed meta tags, and use them!

The web is transient and journalism is vanishing. Media companies act without regard to the historical record, and journalists are losing their work as a result.

*This project is in development, and is not intended for actual, production use.*

## Development
1. Clone repository

2. Install dependencies (with pipenv or environment of choice)

3. Configure script in `config.yaml`
   
4. Run `main.py`
